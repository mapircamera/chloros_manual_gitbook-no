# Utdataformat for bilder

### TIFF (16-bit)

16-bit (piksel digitaltall 0 - 65535) TIFF (.tif) format utdata.

### TIFF (32-bit, prosent %)

32-bit (piksel flytende 0,0 - 1,0) TIFF (.tif) format utdata. Pikselverdier representerer refleksjonsprosentandeler hvor 0,0 = 0 % og 1,0 = 100 %.

### **JPG (8-bit)**

8-bit (piksel digitalt tall 0 -255) JPG (.jpg) formatutdata.
